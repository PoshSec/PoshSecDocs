Get-DomainAdmins
================

Synopsis
--------

Obtains the list of Domain Admins

Description
-----------

Obtains the list of domain admins for the current or specified domain.

Examples
--------
..code::
PS> Get-DomainAdmins
SamAccountName      DisplayName
--------------     -----------
Administrator       
mjadmin             Matt Johnson - Admin

Notes
-----
Part of the PoshSec PowerShell Module
